# 🎓 Depective Review Detection in Education Sector

> Uncovering Deceptive Reviews Using Machine Learning and NLP Techniques with Power BI for Strategic Insights

---

## 📌 Project Overview

This project develops a machine learning model to identify deceptive reviews in the education sector. It combines Natural Language Processing (NLP) techniques for sentiment analysis with an interactive Power BI dashboard to generate strategic insights for educational institutions.

---

## 🎯 Project Objectives

1. Develop a machine learning model that can identify deceptive reviews in the education sector
2. Incorporate NLP techniques to analyze sentiment and patterns within the reviews
3. Utilize Power BI for data visualization and strategic insights generation

| Category | Tool |

## 📈 Results

| Metric | Value |
|---|---|
| Total Reviews Analyzed | 855 |
| Fake Reviews Detected | 118 (13.80%) |
| Genuine Reviews | 737 (86.20%) |
| ML Model Accuracy | 90.84% |
| Positive Sentiment | 722 (84.4%) |
| Negative Sentiment | 61 (7.1%) |
| Neutral Sentiment | 72 (8.4%) |

---

## 🔍 Key Findings

1. **13.80% of education reviews are deceptive** — approximately 1 in 7 reviews is fake
2. **Fake reviews are 97.5% positive** — deceptive reviews use exaggerated positive language compared to 82.4% for genuine reviews
3. **Fake reviews score higher on sentiment** — average sentiment score of 0.32 vs 0.20 for genuine reviews
4. **Tapasya College of Commerce and Management, Hyderabad** had the highest fake review count (13 fake reviews)
5. **Career360 reviews are more positive** (92.8%) compared to Collegedunia (73.5%)
6. **Negative reviews align with lower ratings** — average rating of 2.24 confirming VADER accuracy
